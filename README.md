# Business Registration System

A comprehensive business registration and license management system with separate interfaces for administrators and business owners.

## Features

- User authentication and authorization
- Business registration and management
- License management
- Document upload and tracking
- Admin dashboard with analytics
- Business owner dashboard

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB

## Project Structure

```
business-registration-system/
├── frontend/          # React frontend application
├── backend/           # Express.js backend API
└── README.md         # Project documentation
```

## Getting Started

### Prerequisites

- Node.js 14+
- MongoDB
- npm or yarn

### Installation

#### Backend Setup

```bash
cd backend
npm install
npm start
```

#### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
